# Maria

![Maria Logo](http://windisco.qiniudn.com/MariaBanner.png)

## What's Maria?

an OS X app/widget for aria2 download tool.

## Features

* [x] Download status notification
* [x] Today Widget - DL/UL speed
* [  ] Today Widget - each task progress
* [  ] Low speed mode

## Test Environment

* Xcode 7.3
* Swift 2.2
* OS X 10.11.4
